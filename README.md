# Hey Fitty 🎀

A kawaii-inspired fitness website built with HTML and CSS as a group project for ISTA 329 (Web Development) at the University of Arizona.

## About

Hey Fitty is a multi-page fitness website designed to make working out and eating healthy feel fun and approachable. The site includes workout routines, meal plans, recipes, fitness blog posts, and a contact form.

## Pages

- **Home** — hero section, featured workouts, latest blog posts, nutrition tip
- **Workouts** — gym routines, home workouts, weekly challenge
- **Nutrition** — meal plans, recipes, tip of the day
- **Blog** — fitness tips and motivation posts
- **About Us** — team intro and mission statement
- **Contact** — contact form with name, email, subject, and message

## Built With

- HTML5
- CSS3 (external stylesheet)

## How to Run

1. Clone the repository
```bash
git clone https://github.com/YOURUSERNAME/hey-fitty.git
```

2. Open the project folder in VS Code

3. Install the **Live Server** extension in VS Code

4. Right click `index.html` and select **Open with Live Server**

5. The site opens in your browser at `http://127.0.0.1:5500`
